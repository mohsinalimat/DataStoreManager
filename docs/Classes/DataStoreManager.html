DataStoreManager

@objcMembers
open class DataStoreManager : NSObject

An interface to the data store manager, where you store key-value pairs persistently across launches of your app.

  • object(forKey:withValueType:completionHandler:)

    Returns the object associated with the specified key.

    For CloudKit, the valueType are needed to properly map the array of data. The valueType will be the type of the expected object. For example:

    let manager = DataStoreManager()
manager.read(forKey: "Key", withValueType: String.self) { (object) in
    if let object = object {
        print("successfully read \(object) from Cloud Kit")
    }
}
